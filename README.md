<h1> Proyecto de portafolio modulo 2 - Bootcamp Front-end </h1>
<p> Este proyecto presenta 10 tarjetas que muestran datos acerca del clima actual (para el momento en que se este mirando) obtenidos de la API de openweathermap.com</p> </br>
<p> Al hacer click para ver mas detalles de una de las ciudades, se redirecciona a la pagina de detalles , donde se muestra la temperatura para la semana (maximo 6 dias por limitacion del plan de la API) </p> </br>
<p> Un detalle a mencionar es que la pagina de detalles por si sola no entrega informacion, se debe necesariamente hacer click en alguna de las tarjetas de la pagina de inicio</p>
<p>Los contenedores de texto poseen unicamente texo generado</p>

<p>pagina al repositorio publico: https://github.com/TomLecker/weather-frontend-m2.git </p>
